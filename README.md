# 포켓몬 팀 빌더

포켓몬 메인 시리즈 본편 공략을 위한 팀 구성 도우미 데스크탑 앱입니다.  
어떤 포켓몬을 잡아야 할지, 팀의 약점은 어디에 있는지, 사천왕·관장을 상대로 어떤 포켓몬이 강한지를 한눈에 파악할 수 있습니다.

---

## 주요 기능

### 팀 빌더
- 지역 도감에서 포켓몬을 검색·필터링해 팀에 추가
- 타입·진화 단계·체육관 입수 시기 기준으로 필터
- 팀 슬롯 6개 구성, 버전별 팀 자동 저장·복원

### 추천 시스템
- 현재 팀의 타입 커버리지를 분석해 부족한 포켓몬을 점수 순으로 추천
- 사천왕·챔피언·관장·빌런 트레이너 포켓몬 기준 공격 커버리지 반영
- 팀 약점 대응(내구·저항·공격) 점수, 스탯 보너스, 타입 중복 패널티 포함
- 추천 결과 상세 탭에서 각 점수 항목별 근거 확인 가능

### 엔트리 분석
- 팀 전체의 공격 타입·방어 내성을 요약
- 팀이 커버하지 못하는 약점 시각화

### 장소 순서
- 버전별 장소를 체육관 진행 순서대로 정렬
- 각 장소에서 잡을 수 있는 포켓몬 목록 확인

### 트레이너
- 관장·사천왕·챔피언·라이벌 등 주요 트레이너 포켓몬 구성 확인
- 트레이너별 타입 특화 정보 제공

### 방어 분석
- 팀 합산 약점·내성 집계 (합산 탭)
- 포켓몬별 개별 방어 분석 (포켓몬별 탭)

### 전국 도감
- 통신진화·돌진화·전설·환상·친밀도진화 특수 필터
- 그룹 보기(진화 계보 묶음) / 목록 보기 전환
- 진화 조건, 통신 교환 아이템, 지역 번호 표시

### 도구 도감
- 버전별 도구 획득처·획득 방법 확인
- 태그별 필터 (통신 아이템, 진화의 돌, 화석, 배틀 강화 등)

### 타입 상성표
- 방어 탭: 타입 조합 선택 → 배율 일람
- 공격 탭: 공격 타입 선택 → 방어 타입별 최고 배율

### 기술 탭
- 버전 내 모든 기술 목록 (Z기술·다이맥스 필터)
- 기술 선택 시 해당 기술을 배울 수 있는 포켓몬 목록

### 종족치 탭
- 버전 내 포켓몬 종족치 비교 (번호·스탯별 정렬)
- 다중 선택 시 평균 비교 모드

---

## 추천 알고리즘

추천 점수는 다섯 가지 항목의 합산입니다.

```
총점 = ① 트레이너 커버리지 + ② 약점 대응 + ③ 패널티 + ④ 초반 접근 보너스 + ⑤ 스탯 보너스
```

---

### ① 트레이너 커버리지 점수

현재 팀이 커버하지 못하는 타입을 얼마나 채워주는지 평가합니다.

**베이스 점수** — 팀에 없는 타입 조합(시그니처)에 대해 후보 포켓몬의 공격·방어 성능을 보고 최대 5점을 부여합니다.

| 점수 | 조건 |
|------|------|
| 5점 | 해당 타입에 4× 공격 가능 + 해당 타입에 ½× 이하 방어 |
| 4점 | 해당 타입에 2× 공격 가능 + 해당 타입에 ½× 이하 방어 |
| 3점 | 해당 타입에 4× 공격 가능 |
| 2점 | 해당 타입에 2× 공격 가능 |
| 1점 | 해당 타입에 ½× 이하 방어 (공격 효과는 낮음) |

> **예시 — PT 버전, 팀에 불꽃/물/풀 타입 보유 중**
>
> 팀이 커버하지 못하는 타입 중 "강철" 타입이 있다고 가정합니다.
> - 루카리오(격투/강철): 격투 타입으로 강철에 2× 공격 가능, 강철 타입이라 강철에 ½× 방어 → **5점**
> - 도롱마담(벌레/강철): 강철에 ½× 방어, 하지만 공격 효과 낮음 → **1점**

**트레이너 필터 배율** — 같은 베이스 점수라도, 버전 내 트레이너들이 실제로 많이 사용하는 타입이면 높은 배율을 받습니다. 거의 등장하지 않는 타입이면 배율이 낮아집니다.

| 배율 | 조건 |
|------|------|
| ×1.00 | 관장·사천왕·라이벌 등 중요 트레이너가 보유한 타입 |
| ×0.75 | 일반 트레이너들이 자주 사용하는 타입 |
| ×0.50 | 일반 트레이너들이 드물게 사용하는 타입 |
| ×0.25 | 버전 내 트레이너가 거의 사용하지 않는 타입 |

**보너스** — 관장·사천왕·챔피언의 개별 포켓몬을 보고 추가 점수를 누적합니다.

| 보너스 | 조건 |
|--------|------|
| +0.20 | 해당 포켓몬에 4× 공격 + 해당 포켓몬의 공격을 ½× 방어 |
| +0.15 | 해당 포켓몬에 2× 공격 + 해당 포켓몬의 공격을 ½× 방어 |
| +0.10 | 해당 포켓몬에 4× 공격 |
| +0.05 | 해당 포켓몬에 2× 공격 |

> **예시 (계속)**
>
> PT 사천왕 마지막 멤버(챔피언 난천)의 팀에 강철 타입 포켓몬이 있다면,  
> 루카리오가 그 포켓몬에 4× 공격 + ½× 방어를 동시에 만족하면 → 보너스 **+0.20** 추가 누적.

---

### ② 약점 대응 점수

팀이 약점을 갖는 타입에 대해 후보 포켓몬이 얼마나 잘 버티고 반격할 수 있는지를 평가합니다.  
타입별로 최대 2.00점이며, 팀의 모든 약점 타입에 대해 누적합니다.

세 가지 축으로 판단합니다.

- **내구**: 해당 타입 공격을 버틸 수 있는가 (관장·사천왕 평균 공격력 기준)
- **저항**: 해당 타입 공격을 ½× 이하로 받는가
- **반격**: 해당 타입에 2× 이상으로 공격할 수 있는가

> **예시 — 팀에 얼음 약점이 있을 때**
>
> 팀이 얼음 타입 공격에 약한 상황입니다.
> - 불카모스(벌레/불꽃): 얼음에 내구 충분, 얼음 저항(½×), 불꽃으로 얼음 2× 반격 → 세 축 모두 만족 → **높은 점수**
> - 야느우(물): 얼음을 저항하지 못하고(1×) 반격도 어려움 → **낮은 점수**

---

### ③ 패널티

| 패널티 | 조건 |
|--------|------|
| −1점 | 타입 중복 — 팀에 이미 있는 타입과 1개 이상 겹침 |
| −5점 | 저종족치 — 종족치 합계 380 이하 (최종 진화형에 비해 너무 약한 포켓몬) |
| −1점 | 늦은 입수 — 팀에 커버가 안 된 시기가 있는데 해당 포켓몬은 그 이후에야 잡을 수 있음 |

> **예시**
>
> 팀에 이미 불꽃 타입이 있는데 또 다른 불꽃 타입 포켓몬을 추천 목록에 올리면 **−1점** 적용.

---

### ④ 초반 접근 보너스

팀이 비어 있고 스타팅 포켓몬도 없을 때, 초반에 잡을 수 있는 포켓몬에게 소폭의 보너스를 부여합니다.

---

### ⑤ 스탯 보너스

버전 내 관장·사천왕 전체 포켓몬의 평균 공격력·평균 속도를 기준으로 후보 포켓몬의 스탯을 비교합니다.

| 보너스 | 조건 |
|--------|------|
| +1.0점 | 공격(또는 특공) AND 속도 모두 평균 초과 |
| +0.5점 | 둘 중 하나만 평균 초과 |
| 0점 | 둘 다 평균 미달 |
| −0.5점 | 공격 미달 + 팀보다 빠른 적이 5마리 미만 (속도 강점도 없음) |

> **예시**
>
> 기라티나 버전 기준 관장·사천왕 포켓몬 평균 공격이 100, 평균 속도가 85라고 할 때,
> - 루카리오(공격 110, 속도 90): 둘 다 초과 → **+1.0점**
> - 갸라도스(공격 125, 속도 81): 공격만 초과 → **+0.5점**
> - 해피너스(공격 5, 속도 50): 둘 다 미달, 팀보다 빠른 적도 적음 → **−0.5점**

---

### 점수 예시 종합

> **PT 버전, 팀: 스타팅(불꽃) + 진화형 2마리 구성 중, "루카리오" 추천 시**
>
> | 항목 | 점수 |
> |------|------|
> | ① 트레이너 커버리지 (강철/격투 미커버 타입 커버, 사천왕 보너스 누적) | +6.50 |
> | ② 약점 대응 (얼음·에스퍼 약점 대응) | +1.80 |
> | ③ 패널티 (없음) | 0 |
> | ④ 초반 접근 보너스 (해당 없음) | 0 |
> | ⑤ 스탯 보너스 (공격·속도 모두 평균 초과) | +1.0 |
> | **총점** | **9.30** |

점수가 높을수록 현재 팀의 약점을 잘 채워주고 후반 트레이너를 상대하기에 적합한 포켓몬입니다.
